###  ꒰(˶• ᴗ •˶)꒱ Axotchi

Axotchi is a cozy pastel virtual pet game inspired by classic tamagotchi toys.

Take care of a sleepy little axolotl by feeding it, letting it rest, playing with it, and keeping its mood balanced through different interactions and reactions.

-------------------------------------------------------------------------------------
✦ **Features**


- Pixel-art axolotl animations
- Feeding, sleeping, and play interactions
- Mood-based dialogue bubbles
- Dynamic background tinting based on overall stats
- Day/night UI transitions
- Squish interaction with reactions
- Hover interactions and idle animations
- Cozy pastel pixel aesthetic

-------------------------------------------------------------------------------------
✦ **Built With**

- Godot Engine
- GDScript

-------------------------------------------------------------------------------------
✦ **Download**

✦ **Installation**

1. Download the game `.zip`
2. Extract the folder
3. Run `Axotchi.exe`

If Windows shows a security warning:
- Click **More info**
- Then click **Run anyway**

✦ **Controls**

- Feed button → restores hunger
- Play button → restores happiness
- Sleep button → restores energy
- Click/Squish the axolotl for reactions

-------------------------------------------------------------------------------------
✦ **Project Structure**

```text
Assets/   -> sprites, UI, audio
Scenes/   -> Godot scenes
Scripts/  -> game logic
Font/     -> fonts/UI resources
```
